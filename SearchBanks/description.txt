Визаальная система многокритериального поиска объектов (Объект  - Пункт обмена валют)
Программное обеспечение должно обеспечивать решение следующих задач: 
Отображение на карте объектов с поясняющими данными (наименование, адрес, курс и т.д.).
Поиск объектов по различным критериям.
Поиск по min/max курсу валют
Поиск ближайшего объекта.
Фильтрация объектов по различным параметрам.
Добавление новых объектов, групп объектов.
Редактирование существующих объектов.
